开发
----

- 安装 [node.js](http://nodejs.org/) 和 [npm](https://www.npmjs.org/)

- 全局安装 [grunt-cli](http://gruntjs.com/getting-started)

  ```
  npm install -g grunt-cli
  ```

- 安装项目依赖，项目根目录运行

  ```
  npm install
  ```

- 启动 grunt，项目根目录运行

  ```
  grunt
  ```

- Chrome 加载 `/build/dev/`

- 在 `/src/` 下开发
